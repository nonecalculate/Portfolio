* **Projects/**

  * 협업 및 개인 프로젝트 정리
  * 각 프로젝트별 README 및 영상 링크 포함

  * 특정 기능(인벤토리, 네트워크, AI 등)을 단일 기능 중심으로 정리한 기술 데모
